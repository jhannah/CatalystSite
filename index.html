The elegant MVC framework

Catalyst will make web development something you had never expected it to be: Fun, rewarding and quick.

Development

Catalyst's philosphy is based on the KISS, DRY and TIMTOWTDI principles.

Keeping it simple and small leads to more robust and scalable applications.

The second of these principles allow you to quickly implement new features so you can Release Early, Release Often and thus Get On With It

There Is More Than One Way To Do It means you can choose from a wealth of models, views and plugins, the ones that fit you best. Whether you like DBIx::Class, Class::DBI::Sweet or Class::DBI, whether you prefer Template Toolkit, Mason or HTML::Template, if you want sessions, form validation or authentication, you can have any or all of them.

Catalyst comes with its own test server which allows you to start a lightweight application webserver without having to setup a full webserver for development. On Unix and Linux it will even restart itself when it detects that sources have changed to allow you to see results instantly!

Deployment

You can deploy your Catalyst application on FreeBSD, Linux, Mac OS X, Solaris or Windows.

Do you have a favorite web server? Catalyst can run under IIS, Zeus, Apache, lighttpd either as CGI or FastCGI application. If you use Apache you can also run your application with mod_perl (any version).

With it's wide support of operating systems and application servers, Catalyst allows you to deploy your applications on a system of your choice or adapt existing applications to take advantage of the benefits Catalyst offers. You don't have to adapt to Catalyst because it can adapt to you.

Scalability

Catalyst scales. Catalyst allows you to grow.
